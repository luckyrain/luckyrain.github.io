Project Description

总体规划
Project Description
汇美国际服饰展贸中心，是由株洲市，区两级政府立项的重点建设项目"株洲沪松服饰物流中心"的 商业组团，也是株洲市千亿服饰产业重点配套项目，旨在为株洲市服饰产业升级提质打造平台，其复合了“配送，仓储 ，展示，交易”的综合体功能，为株洲市服饰龙头企业提供了全爷态整合服务，是当今服饰产业的最高业态，也是达州株洲服饰 产业升级版的龙头项目

项目总占地面积200亩，总建筑面积38万平方米，项目业务涵括：

  1. 服饰展贸中心
  2. 服饰发布中心-“株洲小巨蛋”
  3. 国际公寓配套
  4. 物流配送中心（由沪松区政府配套开发）


    5. 整体规划停车位达到2179个，其中包括148个大货车车位。
